# jQuery Exercises

## Structure

```
jquery/
├── 01-library-inclusion/   Exercise 1 — CDN inclusion + Hello World console log
├── 02-selectors/           Exercise 2 — $() selectors, text change, hide on click
├── 03-methods/             Exercise 3 — hide, show, fadeOut, fadeIn, toggle, chained slideUp/slideDown
├── 04-dom-manipulation/    Exercise 4 — form input → append <li>, remove all
├── 05-events/              Exercise 5 — click to red, dblclick to white
├── 06-event-helpers/       Exercise 6 — click, dblclick, mouseenter, mouseleave, keypress
└── README.md
```

## How to Run

Open any `index.html` file directly in a browser. No build step or server required.
Each file loads jQuery 3.7.1 from the official CDN.
